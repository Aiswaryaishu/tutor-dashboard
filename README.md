# Tutor Dashboard — Frontend Developer Assessment
A responsive, pixel-perfect Tutor Dashboard developed as part of the Frontend Developer Assessment for Grapelime Innovations.
Built using React + Vite + TypeScript and Tailwind CSS v4, this project faithfully reproduces the provided UI design while improving functionality and correcting layout inaccuracies.

---
# Setup Instruction
- 1. Extract the Project Unzip the project archive to your preferred location.
- 2. Open the Project Folder Open the folder in your terminal or VS Code:
     cd C:\Projects\tutor-dashboard
- 3. Install all required libraries using npm:
     npm install
---
# How to run in local machine
- 1. Start the Development Server
     npm run dev
- 2. Once the server starts, you’ll see an output like:
     VITE v7.1.7  ready in 500ms
     Local:   http://localhost:5173/
  3. Open this URL in your browser to preview the dashboard.
---
# Technologies and libraries used
- Frontend Framework : React 19
- Language : TypeScript 5.9
- Bundler / Dev Server : Vite 7
- Styling : Tailwind CSS 4.1
- PostCSS : PostCSS + Autoprefixer
- Linting : ESLint 9

# Assumptions made
- The calender dates given in the snapshot were wrong (repeated). It is corrected in the project.
-"Upcoming” and “Completed” sections were reversed in the provided design. On desktop, I aligned them logically, but on mobile, I kept them as per the original design.
- The calendar now supports dynamic months.
